1. Создать репозиторий на Github.
2. Создать репозиторий на локальном компьютере.
3. Инициализировать репозиторий - git init.
4. Добавить файлы в директорию репозитория, перейти в ветку локального репозитория - cd c:/git_rep/tech
и дать команду - git add .,  создать коммит - git commit –m “Начало”.
5. Добавить удалённый репозиторий -
git remote add origin https://github.com/armeryyr/technomart.git
(чтобы посмотреть, какому URL соответствует сокращённое имя в Git, можно указать команде опцию -v: git remote –v)
6. Отправить изменения из локального репозитория на удалённый репозиторий
git push origin master
